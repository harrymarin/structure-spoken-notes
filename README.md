# structure-spoken-notes

A reusable Codex/OpenClaw skill for turning spoken, messy, or chat-style wording into clear structured text without changing the original meaning.

## What It Supports

- Detailed version
- Concise version
- Business-formal version

## Install

Copy this folder into your local skills directory, for example:

```bash
cp -R structure-spoken-notes ~/.agents/skills/
```

## Use Cases

- Rewrite rough notes into clearer writing
- Turn spoken language into structured text
- Keep meaning intact while improving logic and flow
- Produce versions suitable for internal notes or external communication
